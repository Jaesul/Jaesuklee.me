OXO Handheld Screwdriver Prototype

Project Details

OXO, the well-known kitchen equipment company, innovated the industry by designing tools with universality in mind. This prototype takes OXO’s design philosophy and pairs it with a handheld screwdriver.

Research

The first step in creating a prototype based on OXO’s design principals is to know what they are. Smart Design, OXO’s equipment designer, states a simple ethos when designing products.

With Smart’s help, the OXO brand has continued to expand and evolve while remaining true to one simple but powerful idea: create products that make everyday life easier.

This manifests in a few design guidelines

  • Universal Design
    • The product should be accessible to a broad range of users
  • Simplicity
    • Simple black and silver products
  • Making Electric Meaningful
    • Simplify not complicate
    • Give what the user needs, no extra bells and whistles

Designing

I started out by looking at images of existing OXO products in their Good Grips line. I found that nearly all products in this range had these characteristics.

  • Solid large radius ovular handle
  • Solid color (black)
  • Simplictic, nothing special, no gimmicks

I took these physical characteristics and created basic sketches which are shown on the right. I addressed the three characteristics in these ways

Prototyping

I followed the basic sketches to create a physical model of the screwdriver. From where tip of the screwdriver to the base where the battery is stored these are the features.

  • Automatic interchangeable tip
  • Screen indicator for number of screws used
  • Speed selector for drill speed
  • Direction selector
  • Trigger for drilling
  • Screen for selecting drill bit

Initially I had a large head for the tool but after testing I realized it was too bulky. I replaced it with a slimmer head to improve balance and simplicity. I also slimmed down the handle as I tested with users with smaller hands and they were unable to properly grip the screwdriver.

Testing

I had my roommate perform a list of tasks to test the screwdriver. The list of tasks were,

  • Set the screwdriver to screw in
  • Screw in a screw
  • Check the number of screws screwed (say out loud)
  • Change screw bit
  • Change screw speed
  • Set the screwdriver to unscrew
Video for user testing

Takeaways

  • Prototyping is fun
    • This was my first time prototyping and I had a lot of fun watching my design come alive in a physical form
  • Iterations
    • I only made one prototype and one set of sketches. I would have liked to make multiple different versions rather than adjusting
  • Fidelity
    • I should have made a lower fidelity prototype to begin with the one I ended up was too high fidelity to iterate on
  • User Testing
    • Oxo is all about usability. To test if my prototype is usable it would have been better to see how comfortable a wide variety of users handle using the tool
